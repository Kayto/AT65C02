## SMON Machine Language Monitor

**Original Authors:** Norfried Mann and Dietrich Weineck  
**Published:** 64er Magazine (November 1984, December 1984, January 1985)  
**Adaptation:** David Hansel (2023)  
**Source:** https://github.com/dhansel/SMON  
**License:** No explicit restriction (published magazine code)

Original SMON was designed for the Commodore 64. This version has been adapted for minimal 6502 systems with significant modifications for AT65C02 hardware.

## AT65C02 Modifications

**Author:** kayto@github.com  
**License:** MIT License  
**Year:** 2025

Custom modifications include:
- Shell mode integration with SMON monitor
- BRK handler improvements with separate entry points (SMON/SMON_SHELL)
- Enhanced trace command functionality (TW, TB, TQ, TS)
- AT65C02-specific hardware adaptations
- LCD and VIA interface implementations
- trace_test validation program and comprehensive testing documentation
